ОПЕРАЦІЙНІ СИСТЕМИ РЕАЛЬНОГО ЧАСУ ТА 32 БІТНІ  МК
Лабораторна робота 8. Дослідження процесів передачі повідомлень і взаємодії між потоками у операційній системі на прикладі FreeRTOS

Обладнання: STM32F411 Discovery, CubeMX, Keil.
Мета: Ознайомитися з механізмами міжпоточної взаємодії у FreeRTOS, навчитися створювати кілька потоків, 
організовувати передачу повідомлень між ними та реалізувати керування потоками за допомогою кнопок.
______________________________________________
Завдання: Створити чотири потоки для керування світлодіодами PD12, PD13, PD14, PD15. Кожен потік має власну частоту миготіння. 
Кнопки PA0–PA3 змінюють тривалість імпульсу відповідного світлодіода через передачу повідомлення у чергу.
